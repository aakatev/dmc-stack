Hey, CPP Server!

Webserver serving this page build with C++ microframework Crow. It uses other C++ libraries: boost, bsoncxx, and mongocxx.

It supports MongoDB, and Crow framework have its own implementation of Mustache, allowing for backend rendering.

Easy to setup, develop, and deploy with Docker! Learn more on projects GitHub repo.
